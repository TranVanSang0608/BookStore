# AGENTS.md

Toàn bộ hướng dẫn cho AI agent làm việc với repo này (bối cảnh dự án, lệnh dev/test, quy tắc kiến trúc bắt buộc, trạng thái hiện tại) nằm trong **[CLAUDE.md](CLAUDE.md)** — đọc file đó trước khi làm bất kỳ việc gì.

File này cố ý chỉ là con trỏ: trước đây nó là bản copy đầy đủ và đã bị lệch trạng thái so với CLAUDE.md (CLAUDE.md ghi Phase 1 xong, file này vẫn ghi Phase 0). Một nguồn sự thật duy nhất thì không bao giờ lệch.

Lưu ý nhỏ khi đọc CLAUDE.md: mục Preview nhắc tới `.claude/launch.json` — đó là cấu hình preview riêng của Claude Code, agent khác không hỗ trợ thì bỏ qua; các lệnh `npm run dev` thủ công vẫn dùng được như mô tả.
